# CodeOrbit Companion — Brand Spec

来源：产品 brief 固定色板（参考椒盐音乐气质，非外链抓取）。

## Color tokens

| Token | Hex | Role |
|---|---|---|
| `--bg` | `#FFF8F0` | 页面奶油白底 |
| `--surface` | `#FFFFFF` | 卡片 / 输入面 |
| `--fg` | `#2C2416` | 主文字 |
| `--muted` | `#7A6E5F` | 次文字 / 说明 |
| `--border` | `#E8DFD2` | 轻描边 |
| `--accent` | `#3D7A6A` | 主强调 / 已连接 |
| `--accent-soft` | `#E3F1EC` | 强调浅底 |
| `--danger` | `#C45C4A` | 拒绝 / 危险 |
| `--danger-soft` | `#FBECE9` | 危险浅底 |
| `--success` | `#4F8A5B` | 成功 |
| `--warn` | `#C4923A` | 警告 / 离线 |

## Typography

- Display / Body：现代无衬线中文 UI 栈（PingFang SC / Segoe UI / Microsoft YaHei / system-ui）
- Mono：SF Mono / JetBrains Mono — 仅 URL、Token 掩码、相对时间数字

## Posture

1. 圆角 14–18px，软卡片，轻描边，少阴影
2. 大留白、空气感；非霓虹 / 非赛博 / 非厚重 Material 紫
3. 底栏 3 Tab：待处理 / 会话 / 连接
4. 顶栏：左标题，右连接状态 chip + 刷新
5. 禁止：本地 CodeOrbit 服务端启动、Hook 安装、终端激活、桌面多栏、暗黑模式
