## Default Permission

Allow keep-alive FGS and pending alert notifications

#### This default permission set includes the following:

- `allow-keep-alive-start`
- `allow-keep-alive-update`
- `allow-keep-alive-stop`
- `allow-pending-alert-notify`
- `allow-pending-alert-clear`
